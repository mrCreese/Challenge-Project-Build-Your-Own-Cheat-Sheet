CSS properties

Common CSS properties

Property Description Values
color specifies the text color of an element Color values can be hex, rgb, hsl, rgba, hsla, or a color name
background-color specifies the background color Background color values can be hex, rgb, hsl, rgba, hsla, or a color name
border adds a border to an element Borders can vary in width, line style, and color
margin is the spacing outside an element values can be any length value px, %, em, rem
padding is the space between its content and its border values can be any length value px, %, em, rem
display specifies how a certain HTML element should be displayed value inline-block, flex, grid, inherit, block
font-size Specifies the font size of text value px, %, em, rem,property values
text-align Specifies the horizontal alignment of text value left, right, center, justify, initial, inherit
width Sets the width of an element value px, %, em, rem
height Sets the height of an element value px, %, em, rem
font-family Specifies the font family for text value "Times New Roman", Times, serif, Arial, Helvetica, sans-serif
